Deri Prasetyo

Halo, saya seorang freelancer.

Saya adalah freelancer pada bidang desain. Saya telah bekerja di bidang desain buku selama dua tahun terakhir. Saya juga memiliki kemampuan web programming. Saat ini saya sedang menginisiasi project start-up Wacana Belaka.

Tentang Saya

Saya telah menyelesaikan pendidikan Akuntansi Syariah di IAIN Surakarta tahun 2018. Selama kuliah saya memiliki pengalaman menjadi pemimpin redaksi jurnal ilmiah mahasiswa Academica. Selama mengelola jurnal tersebut saya mulai mengenai dunia desain buku serta web programming, sehingga setelah lulus di samping saya memiliki bekal ilmu akuntansi, saya juga memiliki skill lain yakni desain dan web programming. Sampai saat ini, saya masih terus mengembangkan skill tersebut dengan membuat usaha rintisan yang saya namai Wacana Belaka.

Selain mengembangkan usaha rintisan, saat ini saya juga menjadi freelancer. Selama dua tahun terakhir saya berkerja secara serabutan sebagai seorang layouter di berbagai penerbit.
